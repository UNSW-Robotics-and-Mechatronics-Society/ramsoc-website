# Introduction

The events that are displayed on our [website](https://www.ramsocunsw.org) are fetched from Meta's graph API

# Steps

1. Have an exec promote your Facebook account

2. Register for a Facebook Developer Account here: https://developers.facebook.com/apps

3. There are 2 types of tokens that we are interested in:

   - User Access Token

   - Page Access Token
